New Template Example

This is an example to show how new templates are automatically detected!

Just add any HTML file to the templates folder with these meta tags in the head:

  • title: The template name
  • meta name="description": Template description
  • meta name="keywords": Comma-separated tags
  • meta name="icon": Any emoji for the icon
